<task type="auto" tdd="true">
  <name>Task 2: Implement AuthService and AuthController with full register/login/refresh/logout</name>
  <files>
    apps/api/src/auth/auth.service.ts,

  <behavior>
    - register: Creates user in Supabase with Argon2-hashed password; returns 201 + AuthResponse message
    - register duplicate: Rejects with 409 ConflictException when email exists
    - login valid: Verifies password with argon2.verify, generates access (15m) and refresh (7d) JWTs, sets httpOnly cookies, stores hashed refresh token in DB
    - login invalid email: Returns 401 UnauthorizedException
    - login invalid password: Returns 401 UnauthorizedException
    - refresh: Validates refresh token against stored hash, issues new access + refresh tokens, updates stored hash
    - logout: Clears both cookies, nullifies refresh_token_hash in database
  </behavior>
  <action>
    1. Create `apps/api/src/auth/auth.service.ts`:
       - Inject SupabaseService, JwtService, ConfigService
       - `register(dto: RegisterDto)`:
         a. Hash password with argon2.hash(dto.password)
         b. Insert into 'users' table via supabase.from('users').insert({...}).select().single()
         c. Check for error -- if error.code === '23505' (unique violation), throw ConflictException('Email already registered')
         d. Return { message: 'Registration successful' }

       - `login(dto: LoginDto, res: Response)`:
         a. Query user by email: supabase.from('users').select('*').eq('email', dto.email).single()
         b. If no user found, throw UnauthorizedException('Invalid credentials')
         c. Verify password: argon2.verify(user.password_hash, dto.password)
         d. If mismatch, throw UnauthorizedException('Invalid credentials')
         e. Generate tokens and set cookies (extracted to private helper)
         f. Return { message: 'Login successful' }

       - Private `generateTokensAndSetCookies(userId: string, email: string, res: Response)`:
         a. Sign access token: jwtService.signAsync({ sub: userId, email }, { secret: JWT_SECRET, expiresIn: '15m' })
         b. Sign refresh token: jwtService.signAsync({ sub: userId, email }, { secret: JWT_SECRET, expiresIn: '7d' })
         c. Hash refresh token with argon2 and update user's refresh_token_hash in DB
         d. Set access_token cookie: httpOnly, secure (if NODE_ENV=production), sameSite: 'lax', path: '/', maxAge: 15*60*1000
         e. Set refresh_token cookie: httpOnly, secure, sameSite: 'lax', path: '/auth/refresh', maxAge: 7*24*60*60*1000

       - `refresh(userId: string, refreshToken: string, res: Response)`:
         a. Query user by id
         b. If no user or no refresh_token_hash, throw UnauthorizedException
         c. Verify refreshToken against stored hash with argon2.verify
         d. If mismatch, throw UnauthorizedException
         e. Generate new tokens and set cookies (reuse helper)
         f. Return { message: 'Token refreshed' }

       - `logout(userId: string, res: Response)`:
         a. Update user: set refresh_token_hash to null
         b. Clear access_token cookie (set maxAge: 0)
         c. Clear refresh_token cookie (set maxAge: 0, path: '/auth/refresh')
         d. Return { message: 'Logged out' }
