1. Install html-to-image and jspdf in apps/web:
   `cd apps/web && pnpm add html-to-image jspdf`

2. Create `apps/web/src/lib/export-png.ts`:
   - Export async function `exportPng(element: HTMLElement, filename: string): Promise<void>`
   - Uses `toPng` from html-to-image with options: `{ cacheBust: true, backgroundColor: '#ffffff', pixelRatio: 2 }`
   - Creates an anchor element, sets download to `${filename}.png`, sets href to the data URL, clicks it

3. Create `apps/web/src/lib/export-pdf.ts`:
   - Export async function `exportAnalyticsPdf(chartElements: HTMLElement[], teamName: string, range: string): Promise<void>`
   - Uses `jsPDF` in landscape A4 mode ('landscape', 'mm', 'a4')
   - First page: title "${teamName} - Analytics Report", subtitle "Range: ${range} | Generated: ${new Date().toLocaleDateString()}"
   - For each chart element: `addPage()`, capture with `toPng(element, { backgroundColor: '#ffffff', pixelRatio: 1.5 })` (1.5 for PDF to reduce file size per research pitfall 3), add image to page with 10mm margins filling the page
   - Call `pdf.save(...)` with filename `${teamName}-analytics-${range}.pdf`
   - Use pixelRatio 1.5 (not 2) for PDF to keep file size reasonable per RESEARCH pitfall 3

4. Create `apps/web/src/lib/export-csv.ts`:
   - Export function `downloadCsv(csvContent: string, filename: string): void` -- creates Blob with type 'text/csv;charset=utf-8;', creates object URL, triggers download via anchor click, revokes URL
   - Export function `escapeCsvField(value: string | number): string` -- wraps strings in double quotes, escapes internal quotes by doubling them
   - Export function `teamAnalyticsToCsv(data: TeamAnalyticsResponse): string` -- builds multi-section CSV:
     Section 1: "Submission Rates" header row, then Date,Rate (%),Submitted,Total for each submissionRates entry
     Section 2: blank line, "Stress Trend" header, then Date,Low,Medium,High for each stressTrend entry
     Section 3: blank line, "Task Volume by Member" header, then Member,Task Count for each taskVolumeByMember entry
     Section 4: blank line, "Workload Heatmap" header, then UserId,Member,Date,Hours for each heatmap entry
   - Export function `memberAnalyticsToCsv(data: MemberAnalyticsResponse): string` -- builds multi-section CSV:
     Section 1: "Daily Hours" header, Date,Hours,Team Avg
     Section 2: blank line, "Stress Timeline" header, Date,Level
     Section 3: blank line, "Daily Tasks" header, Date,Count
     Section 4: blank line, "Submission Calendar" header, Date,Submitted (yes/no)
   - Import types from '@daily-report/shared'
   - All string fields must be quoted with escapeCsvField to handle commas/quotes in member names
