  <action>
Build the extension and present verification instructions to the user. Run `cd apps/extension && pnpm build` to ensure dist/ is up to date before presenting the test steps.

**Prerequisites:**
- Backend and frontend running: `pnpm dev` (API on :3001, Web on :3000)
- Extension built: `cd apps/extension && pnpm build`
- Have a registered user account with at least one team (from prior phases)

**Load the extension:**
1. Open Chrome, go to `chrome://extensions`
2. Enable "Developer mode" (toggle top-right)
3. Click "Load unpacked", select `apps/extension/dist/`
4. Verify extension icon appears in toolbar (pin it if needed)

**Test 1: Extension Login (EXT-01)**
1. Click the extension icon in toolbar
2. Verify login form appears (email + password fields)
3. Enter your account credentials and click Login
4. Verify: popup switches to task form view, shows your email
5. Close and reopen popup -- verify you are still logged in (tokens persist)

**Test 2: Context Menu on Text Selection (EXT-02)**
1. Navigate to any webpage (e.g., https://example.com)
2. Select/highlight some text on the page
3. Right-click the selected text
4. Verify: "Add to Daily Report" appears in the context menu

**Test 3: Pre-filled Popup (EXT-02, EXT-03)**
1. With text selected, click "Add to Daily Report" from context menu
2. Verify: popup opens with:
   - Title field pre-filled with the highlighted text
   - Source link field pre-filled with the page URL
   - Source link field is editable

**Test 4: Task Creation (EXT-04, EXT-05)**
1. In the popup quick-add form:
   - Title should be pre-filled (from highlighted text)
   - Source link should be pre-filled (page URL)
   - Enter estimated hours (e.g., 1.5)
   - Optionally enter notes
2. Click Save
3. Verify: success message appears in popup
4. Open web app at http://localhost:3000 and navigate to today's daily report
5. Verify: the task you just created appears with correct title, source link, hours, and notes

**Test 5: Direct Popup Use (no context menu)**
1. Click extension icon directly (without selecting text first)
2. Verify: empty task form appears (no pre-filled data)
3. Fill in title, hours manually
4. Save and verify it appears in web app

**Test 6: Logout**
1. Click logout in the extension popup
2. Verify: login form appears
3. Verify: clicking extension icon shows login form (tokens cleared)
