    - Have two users: one manager, one member, both in the same team

    Test 1 - Manager Dashboard Navigation (MGMT-01):
    1. Log in as the manager user
    2. Verify "Manager" link appears in the nav bar
    3. Click "Manager" -- should navigate to /manager

    Test 3 - Pending Submissions (MGMT-02):
    1. On the manager dashboard, check the "Pending" panel

    3. Members with draft reports should show as pending (not submitted)
    4. If all members have submitted, verify "All members submitted" message appears

    Test 4 - CSV Export (MGMT-03):
    1. Click the "Export CSV" button on the manager dashboard
    2. Verify a file downloads named `team-report-YYYY-MM-DD.csv`
    3. Open the CSV -- verify it has the header: Member Name, Email, Date, Report Status, Task, Hours, Source Link, Notes
    4. Verify data rows match what's shown on the dashboard

    Test 5 - Mobile Responsiveness (UI-01):
    1. Open browser DevTools, toggle device toolbar, set to 375px width (iPhone SE)
    2. Verify hamburger menu icon appears (no desktop nav visible)
    3. Tap hamburger -- verify nav links appear vertically with good tap targets
    4. Tap a link -- verify menu closes and page navigates
    5. Check /manager dashboard at 375px -- verify cards stack vertically, tables scroll horizontally
    6. Check /reports page at 375px -- verify task list is usable

    Test 6 - Non-Manager User:
    1. Log in as a regular member (not a manager)
    2. Verify "Manager" link does NOT appear in the nav bar
    3. Navigate directly to /manager -- verify appropriate handling (empty state or redirect)
