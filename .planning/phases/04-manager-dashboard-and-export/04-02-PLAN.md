    2. Create `apps/web/src/app/(dashboard)/manager/[teamId]/page.tsx`:
       - 'use client' page, extract teamId from `useParams()`
       - Date state initialized to today (`format(new Date(), 'yyyy-MM-dd')` from date-fns)
       - Date navigation: prev/next day buttons (same pattern as reports/[date]/page.tsx)
       - Fetch team reports: `useQuery({ queryKey: ['team-reports', teamId, date], queryFn: () => api.get<TeamReportsResponse>(`/teams/${teamId}/reports?date=${date}`) })`
       - Fetch pending: `useQuery({ queryKey: ['pending', teamId, date], queryFn: () => api.get<PendingMember[]>(`/teams/${teamId}/reports/pending?date=${date}`) })`
       - Layout sections:
         a. **Header**: Team name, date with prev/next arrows, Export CSV button
         b. **Pending Submissions panel** (top): Show Badge with count. List pending members by name/email with their status (draft/none). Use amber background for visibility. If empty, show "All members submitted" message with green styling.
         c. **Member Reports section**: For each TeamMemberReport, render a collapsible Card:
            - Card header: member name/email, status Badge (submitted=green, draft=yellow, none=gray), total hours
            - Card content (expandable): table of tasks showing title, hours, source link, notes
            - Use shadcn Collapsible or simple useState toggle for expand/collapse
       - Export CSV button handler:
         ```typescript
         async function handleExport() {
           const API_URL = process.env.NEXT_PUBLIC_API_URL ?? 'http://localhost:3001';
           try {
             const res = await fetch(`${API_URL}/teams/${teamId}/reports/export?date=${date}`, {
               credentials: 'include',
             });
             if (!res.ok) throw new Error('Export failed');
             const blob = await res.blob();
             const url = URL.createObjectURL(blob);
             const a = document.createElement('a');
             a.href = url;
             a.download = `team-report-${date}.csv`;
             a.click();
             URL.revokeObjectURL(url);
             toast.success('Report exported');
           } catch (err) {
             toast.error(err instanceof Error ? err.message : 'Export failed');
           }
         }
         ```
       - Use responsive Tailwind classes throughout:
         - Grid: `grid grid-cols-1 lg:grid-cols-3 gap-4` for pending panel + reports
         - Pending panel: full width on mobile, sidebar on desktop
         - Cards: stack vertically, full width
         - Task tables: use `overflow-x-auto` wrapper for horizontal scroll on mobile
         - Text truncation on long URLs with `truncate max-w-[200px]`

    IMPORTANT: Use `credentials: 'include'` for the CSV fetch (not the api client, since api client expects JSON responses). The raw fetch handles the blob correctly.
