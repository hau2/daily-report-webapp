  <files>apps/web/package.json, apps/web/src/lib/export-utils.ts, apps/web/src/components/analytics/chart-card.tsx</files>
  <action>
1. Install html2canvas and jspdf:
   ```
   cd apps/web && pnpm add html2canvas jspdf
   ```

2. Create `apps/web/src/lib/export-utils.ts` with three exported functions:

   a) `downloadChartAsPng(element: HTMLElement, filename: string): Promise<void>` — Uses html2canvas to render the element to a canvas, then converts to PNG blob via canvas.toBlob() and triggers download with the given filename (append .png). Set html2canvas options: `{ backgroundColor: '#ffffff', scale: 2 }` for crisp output.

   b) `downloadAnalyticsPdf(containerElement: HTMLElement, title: string): Promise<void>` — Uses html2canvas to capture the entire containerElement (the charts area), then creates a jsPDF instance in landscape A4, adds the title as text at the top, calculates image dimensions to fit page width with margins (20mm), adds the captured image, and triggers PDF download. Use `{ backgroundColor: '#ffffff', scale: 2 }` for html2canvas. Name file `${title}-analytics-report.pdf`.

   c) `downloadAnalyticsCsv(data: Record<string, unknown>[], filename: string): void` — Takes an array of flat objects, extracts headers from the first row's keys, builds CSV string with header row + data rows (values joined by commas, strings containing commas wrapped in double quotes), creates a Blob with type 'text/csv', and triggers download. Append .csv to filename.

   Add a shared helper `triggerDownload(blob: Blob, filename: string)` that creates a temporary anchor element, sets href to URL.createObjectURL(blob), sets download attribute, clicks it, and revokes the object URL.

3. Create `apps/web/src/components/analytics/chart-card.tsx` — A wrapper component that replaces direct Card usage for chart containers:

   ```typescript
   interface ChartCardProps {
     title: string;
     children: React.ReactNode;
   }
   ```

   Renders a Card with CardHeader containing the title and a small download icon button (lucide-react `Download` icon, 16x16). Uses a ref on CardContent. On button click, calls `downloadChartAsPng(ref.current, title)`. The button should have `title="Download as PNG"` for accessibility. Style the button as ghost/icon variant, positioned in the CardHeader next to the title using flex justify-between.
  </action>
