Replace hardcoded Tailwind color classes with semantic tokens across all 10 files. Specific replacements:

**Background colors:**
- `bg-white` -> `bg-background` (page/card backgrounds)
- `bg-gray-50` -> `bg-muted` (subtle backgrounds like auth layout)
- `bg-gray-100` -> `bg-muted` (hover backgrounds, subtle fills)

**Text colors:**
- `text-gray-900` -> `text-foreground` (primary text)
- `text-gray-800` -> `text-foreground` (headings)
- `text-gray-700` -> `text-foreground` (body text)
- `text-gray-600` -> `text-muted-foreground` (secondary text like email, labels)
- `text-gray-500` -> `text-muted-foreground` (tertiary text, placeholders)
- `text-gray-400` -> `text-muted-foreground` (disabled/faint text)

**Hover states:**
- `hover:text-gray-900` -> `hover:text-foreground`
- `hover:bg-gray-100` -> `hover:bg-accent`
- `hover:bg-gray-50` -> `hover:bg-accent`

**Border colors:**
- `border-gray-200` -> `border-border`
- `border-gray-300` -> `border-border`
