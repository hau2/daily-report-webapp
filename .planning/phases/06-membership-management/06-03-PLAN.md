  <action>
    Present the verification checklist to the user. Start dev servers with `pnpm dev` if not already running.
    Guide the user through testing each of the 6 MEMB requirements in the browser.

    Prerequisites: Two user accounts (User A as owner, User B as member of a team).

    **Test 1 -- MEMB-04: Cancel Invitation**
    1. Log in as User A (owner)
    2. Go to /teams/{teamId} for a team owned by User A
    3. Send an invitation to a test email via the invite form
    4. Verify the pending invitation appears in the "Pending Invitations" section
    5. Click "Cancel" on the pending invitation
    6. Verify it disappears from the list

    **Test 2 -- MEMB-01: Remove Member**
    1. Ensure User B is a member of User A's team
    2. Log in as User A (owner), go to /teams/{teamId}
    3. Click "Remove" next to User B, confirm the removal
    4. Verify User B disappears from the members list
    5. Log in as User B -- verify the team no longer appears in /teams

    **Test 3 -- MEMB-06: Historical Data Preservation**
    1. After removing User B (Test 2), log in as User A
    2. Go to the manager dashboard for that team
    3. Navigate to a date when User B had submitted a report
    4. Verify User B's historical report is still visible

    **Test 4 -- MEMB-02: Leave Team**
    1. Re-invite User B to the team and have them accept
    2. Log in as User B (member), go to /teams/{teamId}
    3. Click "Leave Team", confirm leaving
    4. Verify redirect to /teams and the team is gone from the list

    **Test 5 -- MEMB-03: Transfer Ownership**
    1. Re-invite User B and have them accept again
    2. Log in as User A (owner), go to /teams/{teamId}
    3. Click "Transfer Ownership" next to User B, confirm
    4. Verify badges swap (User A becomes Member, User B becomes Owner)
    5. Verify User A no longer sees owner-only controls

    **Test 6 -- MEMB-05: Delete Team**
    1. Log in as the current owner
    2. Go to /teams/{teamId}, scroll to "Danger Zone"
    3. Click "Delete Team", confirm deletion
    4. Verify redirect to /teams and the team is gone
    5. Log in as the other user -- verify team is gone for them too
