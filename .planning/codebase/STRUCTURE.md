## Where to Add New Code

Since this is a greenfield project, the structure depends on the chosen framework. Common patterns for a web application named `daily-report-webapp`:

**If using Next.js (App Router):**
- Primary code: `src/app/` (pages and layouts)
- Components: `src/components/`
- API routes: `src/app/api/`
- Utilities: `src/lib/`
- Tests: `__tests__/` or co-located `*.test.ts` files
- Config: `next.config.js`, `tsconfig.json` at root

**If using Next.js (Pages Router):**
- Primary code: `pages/`
- Components: `components/` or `src/components/`
- API routes: `pages/api/`
- Utilities: `lib/` or `src/lib/`

**If using Vite + React:**
- Primary code: `src/`
- Components: `src/components/`
- Pages: `src/pages/`
- Utilities: `src/utils/` or `src/lib/`
- Tests: `src/__tests__/` or co-located

**Regardless of framework, place:**
- Planning docs: `.planning/`
- Environment config: `.env` (root, gitignored)
- Package manifest: `package.json` (root)
- TypeScript config: `tsconfig.json` (root)
